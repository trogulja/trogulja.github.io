## Why a blog?

Every developer needs a corner of the internet to call their own. This is mine.

I built this site using [Astro](https://astro.build) with Tailwind CSS v4 and MDX support. The goal was simple: a fast, minimal site that feels like home for someone who spends most of their day in a terminal.

## The stack

- **Astro 5** — static site generation with island architecture
- **Tailwind CSS v4** — utility-first CSS with the new CSS-based configuration
- **MDX** — markdown with component support for richer blog posts
- **GitHub Pages** — free hosting, deployed via GitHub Actions

## What's next

I plan to write about things I learn, projects I'm working on, and occasionally share opinions about tools and workflows. Nothing fancy — just honest notes from the trenches.
